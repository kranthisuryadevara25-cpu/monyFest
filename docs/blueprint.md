# **App Name**: LoyaltyLeap

## Core Features:

- User Role Management: Assign and manage user roles (superAdmin, agent, merchant, member) using Firebase Auth custom claims.
- Agent Referral System: Generate unique agent codes upon agent sign-up to track referred merchants and members.
- Commission Calculation: Calculate and distribute commissions and rewards based on agent referrals and member purchases. Track transactions.
- Offer Management: Enable merchants to create and manage offers with details like title, description, points, and expiry date.
- Wallet and Transaction Tracking: Implement a virtual wallet system to manage balances, track credit/debit transactions, and distribute rewards.
- Admin Dashboard: Provide superAdmins with tools to manage agents, merchants, members, approve payouts, and send notifications.
- Content Moderation: Implement a tool that allows the superAdmin to leverage generative AI to quickly identify inappropriate or low-quality offers created by merchants.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a sense of loyalty and premium quality.
- Background color: Light gray (#F5F5F5), almost white, providing a clean and modern look.
- Accent color: Cyan (#00BCD4) to highlight interactive elements and important information, creating contrast.
- Body font: 'PT Sans', a modern and humanist sans-serif font, is recommended for its warmth and readability, providing a balanced and engaging user experience. Headline font: 'Space Grotesk' to add emphasis on headings.
- Use consistent, modern, line-style icons to represent different actions and categories throughout the platform. All icons should come from the same font family for visual consistency.
- Employ a responsive, grid-based layout to ensure adaptability across devices and maintain a clean, structured presentation.
- Use subtle animations and transitions to enhance user engagement and provide feedback for interactions (e.g., button clicks, loading states).
